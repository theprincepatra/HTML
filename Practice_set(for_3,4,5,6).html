1: Create a Basic HTML Page Create a new HTML file named practice1.html. Use essential tags: , , , , <body>. Set the page title to: “My First Practice Page”. Inside <body>: Add an <h1> that says “Welcome to HTML Practice”. Add a paragraph describing your experience starting HTML. 2: Add Text Formatting and Highlight Inside that same paragraph: Make one word bold using <b>. Make another word italic using <i>. Underline one word using <u>. Highlight a phrase using <mark>. Add a subscript like H<sub>2</sub>O and a superscript like E = mc<sup>2</sup>. 3: Create a Science Fact Sheet Page Create a file named science.html. Use a page title: “Science Facts”. Inside <body>: Add an <h2> with the text: “Interesting Science Facts”. Write one science fact in a <p> tag. Use <sub> or <sup> for chemical/exponent formats. Highlight the most important line with <mark>. End the paragraph with a small note using <small>. 4: Product Description Page Create a file called product.html. Title the page “Product Description”. Inside <body>: Use an <h1> with the product name. Add a <hr> tag. Add a paragraph describing: Product features with <b> Benefits with <i> Price with <mark> Any trademark/special mark with <sup>™</sup> 5: Simple Welcome Page with Highlights Create welcome.html. Page title: “Welcome Page”. Inside <body>: Add an <h1>: “Hello, Visitor!” Insert a <hr>. Add a paragraph introducing yourself: Make your name <b> bold. Italicize your hobby with <i>. Underline your city using <u>. Highlight your favorite color using <mark>.
